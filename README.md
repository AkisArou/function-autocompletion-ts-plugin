## Typescript language service plugin for function completion for object literals like methods (Experimental)

![Demo](./demo.gif)

## Installation

```sh
npm i -D function-completion-ts-plugin
```

## Usage

```sh
// tsconfig.json
{
  "compilerOptions": {
    "plugins": [{"name": "function-completion-ts-plugin"}]
  }
}
```
